1. Інформація, яку ми збираємо

1.1 Інформація, яку ви надаєте

  • Персональні дані: Для створення персоналізованих гороскопів та прогнозів ми збираємо ваше ім'я та дату народження.
  • Введення даних функцій: Текстові описи снів для Тлумача снів, конкретні запитання для Магічного Оракула та дані про народження для перевірки Сумісності в коханні.
  • Налаштування: Обрана вами мова (англійська, німецька, французька, іспанська, казахська або португальська) та налаштування сповіщень.

1.2 Інформація, що збирається автоматично

  • Технічні дані: Модель пристрою, версія операційної системи та унікальні ідентифікатори пристрою для управління сесіями.
  • Логи інфраструктури: Базові метадані з'єднання, що обробляються через Firebase для хостингу (gertruda.fun) та стабільності роботи додатка.

2. Як ми використовуємо вашу інформацію

Ми використовуємо ваші дані виключно для забезпечення розважальних функцій Додатка:

  • Для створення персоналізованих гороскопів та прогнозів.
  • Для надання результатів сумісності та тлумачення снів.
  • Для керування конфігурацією Додатка через Firebase Remote Config.
  • Для надсилання запланованих сповіщень про повторне залучення (якщо увімкнено).

3. Сторонні сервіси

Ми використовуємо надійні сторонні сервіси для забезпечення технічної функціональності:

  • Firebase (Google): Для інфраструктури додатка, хостингу цільової сторінки та керування віддаленою конфігурацією.
  • Аналітичні API ШІ: Для обробки текстових даних користувача (сни та запитання оракулу) у режимі реального часу для генерації відповідей.
  • Магазини додатків: Apple та Google безпечно обробляють усі транзакції покупок у додатку.

4. Зберігання та безпека даних

Локальне зберігання: Більшість персональних даних, включаючи ваш профіль для гороскопа, зберігаються безпосередньо на вашому пристрої. Ми не ведемо постійної централізованої бази даних ваших приватних описів снів або запитань оракулу.

Хмарна обробка: Хоча вхідні дані обробляються у хмарі для аналізу ШІ, ми надаємо пріоритет ефемерній (тимчасовій) обробці над постійним зберіганням.
